## Projects

### simple_lexer.py
A basic lexer that tokenizes source code into:

- Numbers
- Operators

### mini_lang.py
A tiny language implementation featuring:
- Character-by-character lexer
- Recursive descent parser
- AST generation
- Operator precedence
- Variable assignments using LET
- Expression evaluation
